# Useful Commands

## Rebuild (NixOS)

```sh
sudo nixos-rebuild switch --flake .#<hostname>
```

## Rebuild (darwin)

```sh
darwin-rebuild switch --flake .#<hostname>
```

## Update

```sh
nix flake update
```

## Garbage Collection

```sh
nix-collect-garbage -d
```
